# Ecom Express Sales Dashboard

## Project Overview
This Power BI dashboard analyzes e-commerce sales performance across different product categories, states, and time periods.

## Tools Used
- Power BI
- Excel

## Features
- Revenue Analysis
- Average Order Value (AOV)
- Cancellation Rate Analysis
- Lost Revenue Tracking
- Total Orders Analysis
- Category-wise Revenue
- State-wise Revenue
- Product-wise Revenue
- Quarterly Revenue Trends

## Key Insights
- Identified top-performing products and categories.
- Analyzed revenue contribution by state.
- Monitored cancellation rates and revenue loss.
- Evaluated quarterly sales performance.
